# igame-chinese-hub

本仓库用于归档和发布多个独立 HTML 页面。这些页面不针对任何特定域名或具体网站，仅作为通用页面资料的集合。

## 项目简介

igame-chinese-hub 是一个面向中文环境的 HTML 页面归档仓库。仓库中的页面以独立、自包含的形式存在，便于查阅、分发或嵌入其他项目。所有页面均为纯前端实现，不依赖后端服务。

## 目录说明

```
.
├── pages/           # 存放所有 HTML 页面文件
├── assets/          # 页面引用的静态资源（CSS、JS、图片等）
├── index.html       # 仓库首页/导航页（可选）
└── README.md        # 本文件
```

- `pages/` 目录下每个子目录或文件代表一个独立的页面项目，命名遵循项目内部约定。
- `assets/` 目录用于存放页面共享的资源文件，各页面也可自带资源。
- 首页 `index.html` 若存在，则提供仓库内页面的索引或导航功能。

## 页面归档说明

本仓库中的 HTML 页面以“归档”为主要目的，即保留页面原始形态与功能，不做频繁更新或优化。归档页面可能包含以下特征：

- 使用原生 HTML/CSS/JavaScript 编写，无构建步骤。
- 页面内部所有资源尽量内联或相对引用，以保持独立性。
- 不包含外部跟踪、统计或第三方服务调用。
- 页面功能、样式、交互方式以归档时为准，不保证与未来环境兼容。

### 如何添加新页面

1. 在 `pages/` 目录下创建新文件夹（或以页面名直接创建 HTML 文件）。
2. 将页面所需的全部文件放入对应文件夹。
3. 如有共享资源，可放入 `assets/` 并在页面中相对引用。
4. 提交 Pull Request 或直接推送至仓库（视权限而定）。
